# 🤖 AI Refund Agent

AI-powered refund automation platform with voice interaction, intelligent decision reasoning, and an admin dashboard built using Next.js, Prisma, and Gemini AI.

---

## 🚀 Features

- AI-powered refund decision engine
- Voice input support
- Automatic speech response
- Refund explanation generation
- Chat-based customer interaction
- Admin analytics dashboard
- Decision tracking and logs
- Real-time refund status
- Database integration using Prisma

---

### Frontend

## ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/Sumit-kumar1-stack/ai-refund-agent.git
```

Move into project:

```bash
cd ai-refund-agent
```

Install packages:

```bash
npm install
```

Create environment file:

```env
DATABASE_URL=

GEMINI_API_KEY=
```

Run locally:

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```
